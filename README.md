# DataSpark-Electronics-Retail-Analytics

## Project Overview

DataSpark is an end-to-end retail analytics project developed to analyze business operations for Global Electronics, an international consumer electronics retailer. The project focuses on transforming raw business data into meaningful insights using Python, SQL, and Power BI.

The analysis covers customer behavior, sales performance, product profitability, store operations, and currency exchange rates to support data-driven business decisions. Through data cleaning, exploratory analysis, SQL querying, and interactive dashboards, this project demonstrates the complete data analytics workflow from raw data to business insights.

---

## Project Objectives

The primary objective of this project is to analyze retail data and identify trends that help improve business performance. The analysis is designed to answer key business questions related to customer purchasing behavior, product performance, inventory planning, store efficiency, and international pricing.

## Business Problem

Global Electronics operates across multiple countries and generates large volumes of customer, product, sales, and store data. Without proper analysis, valuable business insights remain hidden.

This project analyzes historical retail data to understand customer preferences, improve sales performance, optimize inventory management, evaluate store operations, and support pricing decisions using currency exchange information.

---

## Business Objectives

### Customer Analytics

* Analyze customer demographics including age, gender, and location.
* Understand purchasing behavior and buying frequency.
* Identify customer segments for targeted marketing strategies.

### Sales Analytics

* Evaluate overall sales performance over time.
* Identify seasonal sales trends.
* Discover top-performing products based on revenue and quantity sold.
* Compare sales performance across stores and regions.

### Product Analytics

* Identify the most and least popular products.
* Analyze product profitability using unit cost and selling price.
* Evaluate category and subcategory performance.

### Store Analytics

* Measure store performance using sales and operational metrics.
* Compare stores across different countries and regions.
* Identify opportunities for business expansion.

### Pricing Analytics

* Analyze the impact of currency exchange rates on international sales.
* Support pricing decisions across global markets.

## Project Workflow

### 1. Data Collection

The project uses multiple datasets containing customer information, product details, sales transactions, store information, and currency exchange rates.

### 2. Data Cleaning

* Removed duplicate records
* Handled missing values
* Standardized data formats
* Converted data types
* Validated data quality

### 3. Data Integration

The datasets were merged using common business keys to create a unified dataset suitable for analysis.

### 4. Exploratory Data Analysis

Performed detailed exploratory analysis to identify trends, patterns, correlations, and business opportunities.

### 5. SQL Analysis

Loaded the cleaned data into SQL tables and developed queries to answer business questions related to sales, customers, products, and stores.

### 6. Dashboard Development

Built interactive Power BI dashboards to visualize key performance indicators and present business insights in an easy-to-understand format.
